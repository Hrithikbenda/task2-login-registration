# Task 2 - Login & Registration UI

## Objective
Create responsive and interactive Login and Registration pages using Bootstrap 5, CSS, JavaScript and PHP.

## Features
- Responsive Login Page
- Responsive Registration Page
- Bootstrap 5 Components
- Custom CSS Styling
- Password Show/Hide Toggle
- Password Validation
- Email Availability Check (Dummy AJAX)
- Dashboard Page
- Hover Effects
- Simple Animations
- Mobile Friendly Design

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
